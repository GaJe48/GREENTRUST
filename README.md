## Hardware
Hardware yang digunakan untuk membangun IoT
* Raspberry Pi 4 model B
* DHT22 (Suhu & Kelembapan)
* Loadcell 20Kg
* Battery Li-Po 5200 mAh
* HX711 (Module Loadcell)
* OLED Display 1,3 inch SH1106
* Step Down XL4015
* HC-SR04 (Ultrasonic)
